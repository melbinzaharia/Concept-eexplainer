The Collection is a framework that provides an architecture used to store and manipulate the group of objects. 
 collection interfaces: The framework has a lot of interfaces that provide just the declaration of methods. 
The framework has the following interfaces:-

Iterator Interface
List Interface
Queue Interface
Deque Interface
Collection Interface
Set Interface
SortedSet Interface

collection classes: All the interface of the collection are implemented by classes and these classes provide body to the method of interfaces
